O que é DOM?

  • DOM significa Document Object Model, ou Modelo de Objeto do Documento.
  • É uma forma de representação do documento (página web).
  • Através do JavaScript conseguimos acessar o DOM e manupular nosso documento.
  • Unir conhecimento do JS com a página para deixa-la dinâmica. Fazer animações, manipular conteúdo. Lado do cliente.
  • É uma forma de representar seu site.
  • Abriu no browser o DOM tá presente.
  • Podemos manipular o próprio documento. O documento é o geral, toda a página, tudo;
  • Node significa nós. Tipo os galhos de uma arvore. Esse é o formato que ficam as tags no HTMS (Uma dentro da outra, fazendo assim uma hierarquia).
  • Elemento de forma idividual, divs, span, p.
  • Quando dizemos "manipular o DOM", basicamente estamos dizendo que o JS vai interagir com o HTML. Por exemplo, quando você utiliza o "innerHTML" para aplicar um texto na tag do parágrafo, "< p> <\p>"
  • DOM é a representação do documento. No DOM temos a arvore dos elementos que compoe o documento.
